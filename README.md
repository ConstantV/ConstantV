# Constant Verweij

Builder, entrepreneur, and AI systems tinkerer based in Scherpenisse, Netherlands.

I work at the intersection of automation, personal infrastructure, and software — building tools that quietly make life and work more intelligent.

Currently running [PMXL](https://constantverweij.nl) and building Metis.

---

## What I'm Building

**[Metis](https://github.com/ConstantV/Metis_New)** — A platform by Maran Solutions for preparing and managing large projects like Turnarounds in the (petro-)chemical industry. Currently in active development for 2026.

**[LifeOS](https://github.com/ConstantV/LifeOS)** — My personal AI operating system built on Obsidian. A digital assistant that learns my workflows, manages knowledge, and surfaces what matters.

**[PAI](https://github.com/ConstantV/claude-dashboard)** — Personal AI Infrastructure. A custom Claude Code setup with skills, agents, hooks, and workflows tailored to how I think and work. PAI was originally created by [Daniel Miessler](https://github.com/danielmiessler) — all credit goes to him.

**[Dagelijkse Briefing](https://github.com/ConstantV/dagelijkse-briefing)** — A daily digest that pulls together weather, calendar events, Todoist tasks, and Apple Reminders into a single morning email.

**[Offline QR Batch](https://github.com/ConstantV/offline-QR-batch)** — A privacy-first QR code generator. No data leaves your machine.

## How I Think About This

Most productivity software is built for the average user. I build for myself — and then share what works.

The goal isn't to be busy. It's to build systems that handle the routine so I can focus on what matters: shipping products, solving real problems, and making better decisions faster with AI as a collaborator.
